{{ _('Welcome to PangyPlot') }}

PangyPlot is a tool designed for visualizing graph-based pangenome data. Currently, PangyPlot is set to visualize data from the HPRC pangenome graph. [Minigraph-Cactus, GRCh38 Graph, clip] with GENCODE annotations.

To quickly run, select an "Example" button at the top of the page. Or input your region of interest then click "Go".

As PangyPlot is still undergoing development, please treat this instance as a public beta. Bug fixes and feature additions are ongoing. If you have a particular region of interest and need help, with visualization, please reach out to scott.mastromatteo@sickkids.ca. You can also report issues on our GitHub Issues page.

For documentation, please visit our Docs page.
